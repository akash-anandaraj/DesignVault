# DesignVault
DesignVault is a Java Swing-based desktop application that helps designers and editors organize and manage digital creative assets.
The application allows users to add, categorize, search, and store project details with persistent data storage using file serialization. It demonstrates strong understanding of Object-Oriented Programming and GUI-based application development.

 Features :
 
 Add new creative assets
 Categorize assets (Logo, Poster, Thumbnail, Video Edit)
Search assets by category
 View all stored assets
 Persistent storage using file handling (Serialization)
 Interactive desktop GUI
 
 Technologies Used :
 
 Java
Java Swing (GUI Development)
Object-Oriented Programming (OOPS)
File Handling (Object Serialization)
Java Collections Framework

Concepts Implemented :

Encapsulation
Layered Architecture (Model & Service separation)
Event Handling
Exception Handling
Persistent Data Management
